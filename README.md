## Requirements

- [node](https://nodejs.org/en/download/)
- [pnpm](https://pnpm.io/installation)

## Usage

### Clone the repository

```bash
git clone git@github.com:cod3rocket/candidates.git && cd ./candidates
```

### Install dependencies

```bash
pnpm
```

### Run the project

```bash
pnpm start
```
